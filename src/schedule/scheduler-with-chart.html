Analytics

Fleet Management Dashboard: A modern timeline scheduler with real-time analytics visualization. Manage driver schedules and monitor fleet metrics through an integrated analytics dashboard.

Overview

This sample demonstrates a professional fleet management solution combining a Timeline Month Scheduler with a real-time Analytics Chart. The left panel displays scheduled routes grouped by driver, while the right panel provides actionable insights through multiple analytical views.

Key Components

  • Timeline Scheduler: Visualizes driver routes and events in a monthly timeline view with resource grouping
  • Analytics Dashboard: Three switchable views for data analysis:
    • Trip Count - Monitor the number of trips per driver
    • Truck Capacity - View truck capacity specifications per driver
    • Longest Trips - Track maximum trip duration in hours
  • Real-time Sync: Chart updates instantly when events are created, edited, or deleted
  • Visual Consistency: Chart bars automatically inherit driver colors from the scheduler

Technical Highlights

  • TimelineMonth view with multi-driver resource grouping
  • Custom resource header and informational tooltip templates
  • Runtime event generation and data binding
  • Color-coded visualization with automatic resource-to-chart mapping
  • Interactive dropdown mode switcher for analytics filtering
  • Responsive design for desktop and tablet devices
  • Modern UI with gradient effects and smooth animations

How to Use

  • View scheduled trips in the timeline on the left
  • Use the dropdown to switch between different analytical views
  • Hover over chart bars to see detailed metrics
  • Create, edit, or delete events to see real-time chart updates
  • Monitor fleet performance across multiple dimensions
