This sample demonstrates the data source binding of the Menu. Mouse hover or touch on the menu items to show it's sub menu pop-up items.

    The Menu component loads the data through the items property, where the data can be either structure as hierarchical data or self-referential data i.e. mapped with the id and parentId fields.

    In this demo, the menu component is bound with the list type data where the parent-child relation is referred by the id and parentId mapping fields.

    More information about Menu can be found in this data binding section.
