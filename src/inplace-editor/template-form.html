This sample demonstrates adding template controls to the In-place Editor component. In this demo, HTML5 controls are showcased with the In-place Editor control.

This sample demos the HTML5 controls using the template property of the In-place Editor control. The controls shown here are:

  • Text
  • Date
  • Select

A template property is bound with an HTML element using its ID as target. Editing the value and updating it back are done with the use of the event actionBegin.
