This sample demonstrates how the syncfusion JavaScript DataGrid, integrated with AI, can predict data based on the grid's existing entries

To explore this and more Syncfusion JavaScript Smart AI integrations locally, check out our GitHub repository.

In this example, the DataGrid displays information such as Student Name and ID, along with GPAs for the first, second, and third years. With AI, the Syncfusion JavaScript DataGrid can forecast the Final-year GPA, Total, and Grade. These predictions are shown in the grid when you click the "Calculate Grade" button
