This sample visualizes the data on US season retail sales growth using a multi-colored area series in the chart. Data points are enhanced with segments and tooltips.

In this example, you can see how to render and configure points in a particular range by using MultiColoredArea series. Points within the range can be configured with the color property in chart segment.

The tooltip is enabled in this example, to see the tooltip in action, hover a point or tap a point in touch enabled devices.


More information on the multi-colored area series can be found in this documentation section.
