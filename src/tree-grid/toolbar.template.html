This sample explains the way of rendering custom template element Refresh in a toolbar and while click on the icon clear the filtering for Task Name column in Tree Grid using API.

Custom toolbar items can be added by defining the toolbar as a collection of ItemModels. Actions for this customized toolbar items are defined in the toolbarClick event.

In this sample, rendered the custom template element Refresh along with predefined toolbar items ExpandAll and CollapseAll. While click on the Refresh button then the filtering is cleared for Task Name column.

More information on the Toolbar Template can be found in documentation section.
