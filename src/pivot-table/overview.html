This sample provides an overview of JavaScript Pivot Table, which allows you to organize and summarize pivot data in a detailed or abstract view and display it as a grid and chart.

The JavaScript Pivot Table is a powerful control for organizing and summarizing business data and displaying the results in a table and chart format. It includes major features such as built-in aggregations, pivot table field list for report manipulation, Excel-like filtering and sorting, editing, Excel and PDF exporting, and so on, which are used in this demo, as well as a large data source loaded without any performance degradation by using virtualization support.
